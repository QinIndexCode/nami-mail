# Nami Mail 0.1.2

[简体中文](v0.1.2.md) | [English](v0.1.2.en.md)

Nami Mail 0.1.2 adds a bilingual interface and controlled mail translation, while refining account setup, message state, and desktop update feedback.

## Download

Download only `Nami Mail Setup 0.1.2.exe` from this Release. Versioned ZIPs, JSON manifests, `latest.yml`, and `.blockmap` are internal release and automatic-update assets, not manual installers.

## What's Changed

- Switch between Simplified Chinese and English. Native notifications, close-to-tray confirmation, and OAuth completion pages follow the selected language. Project locale packs are maintained as JSON files with extension documentation.
- Add optional machine translation for the open message. Configure a LibreTranslate-compatible service in Settings, translate when needed, and select or copy the result. Machine translation can be inaccurate.
- Translation is explicitly opt-in: only the selected message's plain-text body is sent after you choose Translate; the service endpoint and optional API key are encrypted locally.
- Refine account setup and mail state: Chinese IME composition is not interrupted, custom-domain discovery does not steal focus while typing, and archive/trash/unread transitions remain consistent while the server is reconciled, including Gmail All Mail.
- Distinguish a message submitted to the server from one confirmed in Sent, and provide separate recovery guidance for network, TLS, signature/integrity, release-asset, and update-cache-cleanup outcomes.

## Signing and SmartScreen

This installer is not Authenticode signed. Windows SmartScreen can show an unknown publisher or origin warning. Return to the official Release page and verify the repository, tag, installer name, and this note. If anything is unclear, stop rather than bypassing the system warning.

## Automatic Update Status

This Release uses an Ed25519-signed versioned JSON manifest to bind the update ZIP version, name, size, and SHA-512. It is in-app update trust material, not a manual installation file.

Real public-GitHub-Release upgrade verification from `0.1.0` to `0.1.2` is not complete. This note does not describe local builds, CI, manifest signing, ZIP downloads, or the presence of Release assets as verified online automatic updates.

## Known Limitations

- This version provides a Windows x64 installer only.
- Message translation requires an available LibreTranslate-compatible service. Machine-translated text is for reference, and the selected message's plain-text body is sent to that service only after you choose Translate.
- Some mail providers require OAuth, an app password, or a client authorization code. See the email-provider guide for preparation steps.

For more help, see [Windows installation and updates](https://github.com/QinIndexCode/nami-mail/blob/main/docs/INSTALLING.en.md), [email provider setup](https://github.com/QinIndexCode/nami-mail/blob/main/docs/EMAIL-PROVIDERS.en.md), and [privacy and local data](https://github.com/QinIndexCode/nami-mail/blob/main/docs/PRIVACY.en.md).
