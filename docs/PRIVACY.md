# 隐私与本地数据说明

[简体中文](PRIVACY.md) | [English](PRIVACY.en.md)

本文描述当前 Nami Mail 代码路径处理数据的方式，不替代邮箱服务商、OAuth 服务商、GitHub、操作系统或企业管理员的隐私政策。Nami Mail 的设计目标是本地优先：没有由本项目运行的邮件代管后端。

## 数据去向

当前代码会在以下必要场景连接第三方服务：

- 你配置的邮箱服务商：用于 IMAP 收取/同步、SMTP 发送、草稿、标志和附件；
- DNS：用于自定义域名的服务商发现；
- Google 或 Microsoft：仅当你主动使用其 OAuth 登录时，用于授权、令牌刷新和相关邮件权限；
- 你配置的 LibreTranslate 兼容翻译服务：仅当你主动点击邮件正文翻译时，发送当前邮件的纯文本正文；不会自动翻译、自动切换服务或在本地持久化翻译缓存。服务地址、超时和可选 API Key 加密保存在本机；详见[邮件正文翻译说明](TRANSLATION.md)。
- GitHub Releases：仅在已安装、带更新配置且拥有有效信任根的桌面发行版检查或下载更新时使用。更新检查只读取公开 Release 元数据；用户选择更新后才下载版本化 ZIP 和 JSON 清单，不会上传邮箱凭据、邮件内容、本地数据库或 GitHub 访问令牌。

代码库当前没有 Nami Mail 自有的分析 SDK、崩溃上报服务或集中邮件 API。上述描述不意味着第三方服务、网络、代理、安全软件或操作系统不会记录网络元数据；它们各自适用自己的政策。

## 本地保存内容

| 数据类别 | 保存位置和处理方式 |
| --- | --- |
| 账户凭据和 OAuth 刷新令牌 | 使用从主密钥派生的 AES-256-GCM 应用层加密后保存。访问令牌只在运行内存中保存。 |
| 邮件敏感载荷 | Message-ID、主题、地址、摘要、正文、回复链和附件元数据保存在加密载荷中。 |
| 发件队列和出站附件 | 主题、收件人、正文、诊断详情和附件内容使用独立用途的派生密钥加密。 |
| 收件附件内容 | 不复制进本地 SQLite 数据库；用户下载时从邮箱服务商受控流式读取。 |
| 运行元数据 | 账户邮箱地址/服务商、文件夹、UID、时间、标志、大小、投递状态、记录标识、普通设置和背景图片等可能以明文保留。 |

Windows 桌面版的主密钥由 Electron `safeStorage` 使用当前 Windows 用户的 DPAPI 保护，文件位于用户数据目录的 `data/master.key.dpapi`。桌面版不会回退到新的明文主密钥。开发用命令行服务不同：它根据 `MASTER_KEY_PATH` 使用独立开发数据路径，不能与桌面 DPAPI 保护等同。

这不是 SQLCipher 或全盘加密。应用层加密可降低静态文件被直接读取或离线复制造成的风险，但不能防御管理员、同一 Windows 用户下能调用 DPAPI 的恶意进程，或已经解锁并运行中的应用。

## 本地与界面保护

- 本地服务默认绑定 `127.0.0.1`；桌面版使用动态回环端口和每次启动重新生成的本地 API 访问令牌。
- 本地 API 响应采用 `no-store`，桌面启动时会清理可能遗留邮件响应的 HTTP/Service Worker 缓存。
- 邮件 HTML 展示前会清理脚本、表单、嵌入内容和图片等不受信任内容；这不是对邮件安全性的绝对保证。
- `.gitignore` 排除 `.env`、数据库、密钥、邮件导出、日志和发布输出。忽略规则不能替代提交前的人工检查。
- 已下载的更新 ZIP 暂存在当前 Windows 用户的应用数据目录；选择跳过该版本会移除对应缓存。成功安装后，更新助手会对对应 ZIP 和临时展开目录进行有限重试与存在性复核；若仍有残留，新版本启动时只会在该目标版本目录中再次尽力清理，并显示状态。系统或同一用户的其他进程仍可能锁定或改名文件，因此不应把该缓存目录当作长期备份位置，也不能将清理表述为绝对保证。

## 数据控制与备份

- 移除账户、丢弃草稿和删除本地数据时，请先理解界面提示的本地/远端影响。Nami Mail 不应删除邮箱服务商上的邮件，除非你在应用中明确执行对应的远端邮件操作。
- Windows 卸载默认保留用户数据；选择删除数据会删除 `%APPDATA%\Nami Mail`。请在操作前备份必要数据。
- DPAPI 保护与当前 Windows 用户上下文绑定。备份或迁移数据目录时，需要同时考虑原 Windows 账户是否还能解锁受保护主密钥。

如果发现与本说明不一致的数据外发、存储或权限行为，请按 [安全策略](../SECURITY.md) 私下报告。
