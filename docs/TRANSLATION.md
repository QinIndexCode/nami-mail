# 邮件正文翻译

[简体中文](TRANSLATION.md) | [English](TRANSLATION.en.md)

Nami Mail 的邮件翻译是可选功能。它只在你在阅读邮件时明确点击“翻译”后执行，不会在收取、同步、打开邮件、切换界面语言或后台刷新时自动发送邮件内容。

## 配置翻译服务

Nami Mail 使用兼容 LibreTranslate `POST /translate` 协议的服务。已安装的 Windows 桌面版可在“设置 > 翻译服务”中填写服务地址、可选 API Key 和超时；保存后立即生效。保存需要有效的服务地址；如需停止使用本机配置，请选择“移除翻译服务”并在确认窗口中完成删除。该操作会删除此设备保存的服务地址和 API Key；若启动环境提供了配置，删除后会回退到该配置。如只需移除已保存的 API Key，可单独选择“移除已保存的 Key”；该操作会再次确认，且不会保存正在编辑的服务地址或超时。API Key 不会再次显示，并会和本地邮件数据一样加密保存。在 Windows 上，数据库主密钥由 Electron 的 DPAPI 保护。

启动进程环境或开发环境项目根目录 `.env` 仍可提供初始/部署配置：

```dotenv
NAMI_MAIL_TRANSLATION_ENDPOINT=https://translate.example.com/translate
# 仅在服务商要求时填写
NAMI_MAIL_TRANSLATION_API_KEY=
# 1000-60000；默认 25000
NAMI_MAIL_TRANSLATION_TIMEOUT_MS=25000
```

- `NAMI_MAIL_TRANSLATION_ENDPOINT` 必须是没有查询参数、片段、用户名或密码的完整 `/translate` URL。只允许 HTTPS，或本机回环 HTTP，例如 `http://127.0.0.1:5000/translate`、`http://localhost:5000/translate`。
- `NAMI_MAIL_TRANSLATION_API_KEY` 是可选项，只会在你的服务要求时作为 `api_key` 字段发送。不要将它提交到仓库、日志或截图中。
- `NAMI_MAIL_TRANSLATION_TIMEOUT_MS` 的有效范围为 1000 到 60000 毫秒；未设置时为 25000 毫秒。环境配置会在启动时读取；用户随后在设置中保存的本机配置优先并立即生效。

已安装的 Windows 桌面版仍可从 `%APPDATA%\Nami Mail\nami-mail.env` 读取端点和超时作为启动配置。该文件不能保存 API Key；请使用“设置 > 翻译服务”保存 Key，或由受管理的启动环境提供。不要将 API Key 提交到仓库、日志或截图中。

服务应接受类似下面的 JSON 请求，并返回至少含有 `translatedText` 字符串的 JSON：

```json
{
  "q": "邮件的纯文本正文",
  "source": "auto",
  "target": "en",
  "format": "text"
}
```

目标语言由当前界面语言转换为服务的基础语言代码，例如 `zh-CN` 使用 `zh`。服务商要求 API key 时，请按其 LibreTranslate 兼容协议处理 `api_key` 字段。

## 数据与控制

点击翻译后，Nami Mail 才会在本机解密当前邮件的纯文本正文，并将该正文发送到你配置的翻译端点。主题、账户地址、附件、附件内容和本地数据库不会因翻译功能而自动上传。

翻译请求不会自动重试为其他服务，也不会在本地数据库、文件或独立缓存中保存翻译结果。服务配置会加密保存在本机，翻译结果不会缓存；关闭阅读视图、刷新或重启应用后需要再次主动翻译。演示模式不会连接任何翻译端点，而是显示本地确定性的预览结果。

将正文发送到第三方翻译服务前，请确认该服务的隐私政策、数据处理位置和企业合规要求。邮件可能包含敏感信息；即使使用 HTTPS，Nami Mail 也无法替代你对所选服务商的安全评估。

## 结果与故障处理

机器翻译可能误译术语、日期、否定、姓名、代码片段或格式。它只用于辅助阅读，不应作为法律、财务、医疗、安全或业务决策的唯一依据；重要内容请核对原文。

未配置端点、正文为空或超过 50000 个字符时，Nami Mail 会保留原邮件并显示具体错误。对于翻译服务连接，它会区分 TLS 证书验证失败、TLS 协商失败、DNS 无法解析、网络不可达、连接被拒绝、连接中断和超时；TLS 证书错误时不要关闭证书验证，应检查系统时间、代理和服务端证书。

翻译服务返回 401/403/407 时，请检查 API key、访问权限或代理认证；429 表示服务限流；5xx 表示服务暂时不可用。Nami Mail 不会把翻译服务的响应正文展示给界面。检查端点是否以 `/translate` 结尾、网络/代理策略、服务 API key 和超时设置后，再手动重试。
